Run AI Assistant

Endpoint:
URL: https://sip.launchlemonade.app/api/1.1/wf/run_assistant

Method: POST

Authentication:
This API uses Bearer Token authentication. Include the token in the header of your request.

Header:

Authorization: Bearer <your_token_here>
Copy
Replace {your_token_here} with your actual API token which you can find in the Profile section

Request Body:
The request body must be JSON formatted. The following fields are required:

assistant_id (string): Unique identifier for the assistant.
conversation_id (string): Unique identifier for the conversation. This can be left empty if starting a new conversation.
input (string): The user input or query to be processed by the assistant.
Example:

{
    "assistant_id": "1767434915812x581745928837005300",
    "conversation_id": "",
    "input": "Your Input"
}
Copy
Response Format:
The response will also be JSON formatted. The following fields are included in the response:

Conversation_ID (string): Unique identifier for the conversation.
Response_ID (string): Unique identifier for the response generated by the assistant.
Error (string): Indicates if there was an error during the request ("Yes" or "No").
Error_Reason (string): Provides the reason for the error if any occurred.
Example:

{
    "Conversation_ID": "945491230",
    "Response_ID": "0251680630800",
    "Error": "No",
    "Error_Reason": ""
}
Copy
Sample Python Code:
Below is a Python example using the requests library to send a POST request to the API.

import requests
import json

url = 'https://sip.launchlemonade.app/api/1.1/wf/run_assistant'
headers = {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer Key'
}
data = {
    "assistant_id": "1059700",
    "conversation_id": "",
    "input": "Input"
}

response = requests.post(url, headers=headers, data=json.dumps(data))
print(response.json())
Copy
Ensure you have the requests library installed in your Python environment:

pip install requests
Copy
This code sends a POST request to the API with the necessary headers and data, and prints the JSON response.

Get AI Assistant Response

Endpoint:
URL: https://sip.launchlemonade.app/api/1.1/wf/get_run_assistant

Method: POST

Authentication:
This API requires an authentication token to be included in the header of the request. The token should be provided as a Bearer token.

Header: Authorization: Bearer {your_token_here}

Replace {your_token_here} with your actual API token which you can find in the Profile section.

Request Body:
The request body must be JSON formatted. Below is the required field in the request body:

response_id (string): The unique identifier for the response you are requesting.
Example JSON Request Body:

{
    "response_id": "17150"
}
Copy
Response Format:
The response will be in JSON format containing a single key Response which holds a string value. The string includes detailed information formatted in Markdown to enhance readability and presentation.

Example JSON Response:

{
    "Response": "Response."
}
Copy
Sample Python Code:
Below is a sample Python code using the requests library to make a POST request to the API:

import requests

url = "https://sip.launchlemonade.app/api/1.1/wf/get_run_assistant"
headers = {
    "Content-Type": "application/json",
    "Authorization": "Bearer Key"
}
data = {
    "response_id": "51680630800"
}

response = requests.post(url, json=data, headers=headers)
print(response.json())
Copy
This code sends a POST request to the API endpoint with the necessary headers and data, and prints the JSON response.
